# Docs Guide

`docs/` 目录用于放置项目说明、架构文档和历史归档。

## 当前结构

- `architecture.md`：整体架构说明
- `domain-model.md`：领域模型说明
- `event-model.md`：事件模型说明
- `module-specs.md`：模块级规格
- `archive/`：历史文档归档

## 为什么做归档

根目录现在优先保留“仓库入口级”文档：

- `README.md`
- `LICENSE`
- `CONTRIBUTING.md`
- `SECURITY.md`
- `CODE_OF_CONDUCT.md`

一些历史规划或旧版说明仍然保留，但移动到 `docs/archive/`，避免顶层噪音过多。
