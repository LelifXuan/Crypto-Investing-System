# Security

This repository is intended for local research and open-source reference use.

## Reporting

If you find a security issue, please avoid posting exploit details in a public issue.
Open a private report through the repository security reporting feature or contact the maintainer directly before public disclosure.

## Notes

- The default configuration is local-only and single-user.
- Do not commit `.env`, API keys, local secret files, or database snapshots.
- Re-enable proper authentication before exposing the app outside localhost.
