# 事件源优先级矩阵

| 层级 | 代表源 | 主要用途 | 轮询建议 | 备注 |
|---|---|---|---:|---|
| Tier 0 | 国家统计局 / 央行 / 外汇局 / 中国政府网 | 宏观数据、政策、监管 | 5-60 分钟 | 高可信，低频 |
| Tier 1 | 金十 / 东方财富 | 全球宏观日历、休市、会议 | 5-10 分钟 | 聚合日历，适合 fallback |
| Tier 2 | Gate Announcements / Listings / Calendar | 上线、下线、维护、解锁、项目事件 | 2-10 分钟 | 与交易链路最相关 |
| Tier 3 | PANews / Odaily | 快讯、热点、补充说明 | 3-15 分钟 | 噪音更高，必须去重 |
| Tier 4 | Tokenview API | 链上异动、地址监控、交易监控 | 1 分钟或 webhook | API Key 管理必做 |

## 推荐告警分流

- `critical`
  - OMO 规则变化
  - LPR
  - 交易所下架/维护
  - 高权重代币解锁
- `high`
  - PMI/PPI/GDP/国民经济运行
  - 新现货/新合约上线
  - 大额链上异动
- `medium`
  - 一般政策解读
  - 媒体快讯
  - 品牌/论坛活动
- `low`
  - 一般宣传类内容
